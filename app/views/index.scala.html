Scala/Play/Slick example for CRUD operations. Use curl to POST/GET/PUT/DELETE:

Examples

Create a user

curl -H "Content-Type: application/json" -X POST
    -d '{"firstName":"Scala","lastName":"Player","emailAddress":"test@@example.com","password":"xyz"}'
    http://localhost:9000/user

Update a user

curl -H "Content-Type: application/json" -X PUT
    -d '{"firstName":"UpdatedScala","lastName":"Player","emailAddress":"test+update@@example.com","password":"newpass"}'
    http://localhost:9000/user/1
